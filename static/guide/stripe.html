Stripe 決済ガイド

オンライン決済の設定方法をご案内します

1Stripeアカウント作成
  1. Stripe公式サイトにアクセスしアカウントを作成
  2. 本人確認書類をアップロード（運転免許証 / マイナンバーカードなど）
  3. 銀行口座情報を登録
  4. 審査完了後、本番環境が利用可能に
審査には通常1〜3営業日かかります。その間はテストモードで動作確認できます。
2APIキーの取得

Stripeダッシュボード → 開発者 → APIキー から以下を取得してください。

キー種別用途
公開可能キー（pk_）フロントエンド（決済フォーム）
シークレットキー（sk_）バックエンド（サーバー側）
Webhookシークレット（whsec_）イベント受信の署名検証
シークレットキーは絶対に公開しないでください。
3決済リンクの作成（簡単）

コーディング不要で決済ページを作成できます。

  1. Stripeダッシュボード → 決済リンク → 新規作成
  2. 商品名・金額を設定
  3. 生成されたURLをLINEやメールで共有
4Webhook設定（自動化連携）

決済完了時にAidenへ自動通知するためのWebhook設定です。

  • ダッシュボード → Webhook → エンドポイントを追加
  • リッスンするイベント：payment_intent.succeeded
  • Webhookシークレットを環境変数に設定
