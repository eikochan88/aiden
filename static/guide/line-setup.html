LINE セットアップガイド

LINE公式アカウントとBotの初期設定手順

1
LINE Developersへのアクセス
  1. LINE Developersコンソールにログイン
  2. プロバイダーを作成（または既存のものを選択）
  3. 新規チャネル → Messaging API を選択
2
チャネルの設定
設定項目内容
チャネル名アカウント名（公開される）
チャネル説明サービスの簡単な説明
大業種 / 小業種ビジネスに合ったカテゴリ
メールアドレス管理者のメールアドレス
3
アクセストークンの取得
  1. チャネル設定 → Messaging API設定タブを開く
  2. 「チャネルアクセストークン（長期）」を発行
  3. トークンをコピーし、安全な場所に保管
チャネルシークレットとアクセストークンの2つが必要です。
4
Webhook URLの設定
  1. Messaging API設定 → Webhook設定
  2. Webhook URL欄に以下を入力：
    https://aiden-1e0e.onrender.com/webhook
  3. 「検証」ボタンで疎通確認
  4. 「Webhookの利用」をオンにする
5
応答設定
  • 応答メッセージ：オフ（Bot側で制御するため）
  • Webhook：オン
  • あいさつメッセージ：任意
設定完了チェックリスト
  • チャネルアクセストークン取得済み
  • チャネルシークレット取得済み
  • Webhook URL設定・検証済み
  • テストメッセージで動作確認済み
